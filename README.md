# yixiao-uniapp-project
uniapp实战项目，有用户登录、注册、文章信息添加查阅


# 执行
1、安装Hbuilder X https://www.dcloud.io/hbuilderx.html
2、下载代码解压到某文件
3、在Hbuilder X 中直接打开项目 文件--》导入--》从本地目录导入
4、在Hbuilder X 中运行项目（需先配置微信开发者工具或连接手机） 运行--》指定模拟器等
